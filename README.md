# Facilitation Game: Who is the Social Entrepreneur?

This project is a game for our Social Entrepreneurship facilitation.

## Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/rej100/social-facilitation-game.git
   cd social-facilitation-game
   ```
2. **Create and activate a virtual environment**
   ```bash
   python -m venv venv
   source venv/bin/activate
   ```
3. **Install dependencies**
   ```bash
   pip install streamlit pandas matplotlib
   ```
4. **Launch the game**
   ```bash
   streamlit run Facilitation-game.py
   ```
